Dashboard Admin

Selamat datang di halaman admin RobuxRadit. Halaman ini digunakan untuk memantau pesanan Robux, mengecek link gamepass dari pembeli, dan melakukan konfirmasi pesanan yang telah selesai diproses.

Alur Proses Pesanan

Konsep Transaksi

  1. Pembeli membuat gamepass di Roblox.
  2. Pembeli melakukan order Robux ke admin melalui website.
  3. Pembeli memasukkan username akun Roblox dan link gamepass.
  4. Pembeli mengirim pesanan lalu menunggu proses admin.
  5. Admin membeli gamepass yang dibuat pembeli sebagai cara pengiriman Robux.
  6. Setelah pembelian berhasil, admin mengonfirmasi pesanan.
  7. Pembeli menyelesaikan pembelian setelah Robux masuk.
